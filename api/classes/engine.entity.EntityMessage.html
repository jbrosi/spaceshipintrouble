engine.entity.EntityMessage Class

Defined in: js\engine\entity\entityMessage.js:2

Basic EntityMessage which gets send between entities and scripts

Note: Instances of this type should be immutable in most cases. Do not reuse messages - create new messages instead!

Properties

Identifier: The "Messagetype" you may listen to. For example: "collision:entity" Message: Object containing various msg data. Structure depends on message type Sender: The entity which send this message (may be null if there was no entity sending this message).

  • sender engine.entity.Entity

    (optional) the entity sending this message. Might be null

  • isConsumable Boolean

    (optional, default: true). Set to false if you don't want this message to be consumable
